CalCOFI Match DuckDB-WASM

Query 75 years of California Current oceanography in your browser — no install, no credentials, no server. The three retired Plumber-API bio↔env endpoints (ichthyodata, itis_ichthyodata, zooplankton_biomass), plus a free-form SQL shell, all running on DuckDB-WASM + public CalCOFI Parquet on GCS. The emitted SQL is character-identical to calcofi4r::cc_match_*()'s attr(d, "sql") — see the Matching Helpers chapter for the story.
